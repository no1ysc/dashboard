## GET API
elasticsearch의 내용 확인
```
GET sc_twitter/_doc/1
  
GET sc_twitter/_doc/2
```
  
## DELETE API
elasticsearch의 document 삭제
```
DELETE sc_twitter/_doc/1
```
  
elasticsearch의 index 삭제
```

```
